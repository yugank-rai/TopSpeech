# TopSpeech - Speech Therapy App

A comprehensive mobile-first speech therapy application designed to help users improve their pronunciation and speech clarity through interactive lessons and exercises.

## Features

### 🏠 Dashboard
- Personalized greeting based on time of day
- Quick stats overview (streak, XP, accuracy)
- Today's lesson recommendations
- Easy navigation to profile and settings

### 📚 Interactive Lessons
- **R Sound Mastery**: Practice the R sound with listening, selection, and mirror exercises
- **TH Sound Practice**: Learn proper TH pronunciation
- **Breathing Exercises**: Controlled breathing for better speech control
- **Phrase Practice**: Apply skills to full phrases

### 👤 Profile & Progress
- Detailed user statistics
- Achievement system with unlockable badges
- Progress tracking across multiple sessions
- Level progression system

### ⚙️ Settings
- Audio customization (speed, effects)
- Difficulty adjustment
- Notification preferences
- Accessibility options

### 🏆 Achievements
- Streak-based rewards
- Skill mastery badges
- Progress milestones
- Motivational notifications

## Technical Features

- **Responsive Design**: Optimized for mobile devices with touch interactions
- **Progressive Web App**: Installable on mobile devices
- **Smooth Animations**: CSS transitions and transforms for engaging UX
- **Accessibility**: Screen reader support and keyboard navigation
- **Offline Support**: Core functionality works without internet

## Getting Started

1. Open `index.html` in a modern web browser
2. Start with the home dashboard
3. Complete daily lessons to build your streak
4. Track progress in your profile
5. Unlock achievements and level up

## Browser Support

- Chrome 80+
- Firefox 75+
- Safari 13+
- Edge 80+

## Development

The app is built with:
- HTML5 for structure
- CSS3 for styling and animations
- Vanilla JavaScript for interactivity

No build tools or frameworks required - just open the files in a browser!
